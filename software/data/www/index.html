HTH

Hackers Teaching Hackers presents this badge as an offensive and defensive platform for wireless interaction. This badge is powered on an ESP32 and an ATMega32 which provides access to Wi-Fi, Bluetooth, LoRa, and mouse emulation. A variety of tools are provided to you but you are encouraged to build, learn and create your own features.

In the menu to the left you can find access to HTH Net. This is a proof of concept chat application that takes advantage of the LoRa chip installed on your badge. Using 915Mhz you can legally transmit messages over a long distance using the low voltage supplied with just this badge. We ask you remain respectful when utilizing this feature.

At the time of this revision some of the HID capabilities are not fully functional, however we have included the mouse jiggler feature. This is automatically enabled when USB is connected and will move your cursor a small amount to emulate user activity.

Source code and hardware files will be made available at https://github.com/syn-ack-zack
